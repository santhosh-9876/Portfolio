# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-05

### Added
- Initial release of portfolio website
- Responsive navigation with mobile hamburger menu
- Hero section with animated SVG
- About section with skills grid and experience timeline
- Projects section with modal details view
- Blog placeholder page
- Contact form with contact information
- Footer with social links and resume download
- Dark mode toggle with persistent preference
- Smooth animations using Framer Motion
- Full accessibility support (ARIA labels, keyboard navigation, skip links)
- SEO optimization (meta tags, Open Graph, sitemap, robots.txt)
- Tailwind CSS for styling
- ESLint and Prettier configuration
- GitHub Actions CI/CD pipeline
- Deployment configurations for Netlify and Vercel
- Comprehensive documentation (README, DEPLOYMENT guide)

### Features
- Mobile-first responsive design
- Reduced motion support for accessibility
- Sticky navigation with scroll effects
- Animated project cards with hover effects
- Form validation (UI only)
- Social media links
- Downloadable resume

### Technical Stack
- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion 12
- React Router DOM 7
- ESLint & Prettier
